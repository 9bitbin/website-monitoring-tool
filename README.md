# Website Monitoring Tool

A simple website monitoring tool built with Python and FastAPI.

## Features
- Monitors one or more websites
- Checks site status (UP/DOWN)
- Logs results to files
- Optional Discord alerts
- Interactive API dashboard

## Requirements
- Python 3.8+
- Internet connection
